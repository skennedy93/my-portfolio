PropertEAZY

A property management application built with React, Node.js, express, SQLite, PostgreSQL, and Material UI. Payments from tenants are made through Stripe Connect. Includes file uploading via Uppy, Transloadit and Google Cloud Storage so that contracts and photos of maintenance needs can be sent between tenant and owner. Design is responsive to mobile, tablet, and desktop views.
